# AI Developer Portfolio

A modern, interactive portfolio website showcasing AI development, chatbot building, and automation projects.

## Features

- **Neural Network Background**: Animated canvas with interactive nodes and connections
- **Responsive Design**: Optimized for all devices
- **Custom Cursor**: Unique cursor effects with hover states
- **Smooth Animations**: Scroll-triggered reveals and typewriter effects
- **Project Showcase**: Interactive cards with live demos and GitHub links
- **Contact Integration**: Direct links to social profiles and contact methods

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (CDN), Custom CSS
- **Fonts**: Google Fonts (Instrument Serif, Cabinet Grotesk, Geist Mono)
- **Icons**: Custom favicon set
- **Animations**: Canvas API for neural network, CSS transitions

## Development

To run locally:

1. Clone the repository
2. Open `index.html` in your browser, or
3. Use a local server: `python -m http.server 8000`

## Deployment

This site is designed to be deployed on GitHub Pages:

1. Push to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Select the main branch as source
